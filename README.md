# Ping – Single Column Coming Soon Page

## A minimal coming soon page with email notification signup

This project implements a single-column “coming soon” landing page for Ping, designed to clearly communicate the launch message and capture user interest through an email notification form. The layout includes a logo, announcement text, and a simple email input to allow users to subscribe for updates.

The page focuses on clarity, responsive behavior, and form interaction, ensuring a clean and accessible experience across different screen sizes.
